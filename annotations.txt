233. ValidationPipe e class-validator para validação dos dados da request no NestJS

Dependências para instalar:
npm i class-validator class-transformer

[arquivo main.ts]:

app.useGlobalPipes(
  new ValidationPipe({
    whitelist: true,
  }),
);

A propriedade 'whitelist' elimina todas as propriedades das classes DTO que não
possuem um 'decorator'. Por isso, deve-se evitar propriedades sem 'decorator'.

[arquivo auth/dto/login.dto.ts]:

import { Type } from 'class-transformer';

@Allow()
@Type(() => Number)
foo: number;

O 'decorator' @Type converte o valor recebido do corpo da requisição no tipo 'Number'.
